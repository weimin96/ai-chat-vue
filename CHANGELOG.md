# Changelog

本文件记录 `@weimin96/ai-chat-vue` 的重要变更。

格式遵循 Keep a Changelog，版本号遵循 Semantic Versioning。

## 0.0.1 - 2026-05-05

### Added

- 首次发布 Vue 3 AI 聊天组件库。
- 提供聊天容器、消息气泡、输入框、欢迎页、推理过程、工具调用、代码块、Markdown 和 Mermaid 等组件。
- 提供 headless composables 和 renderless 组件，支持自定义样式接入。
- 提供 VitePress 文档站和 play 本地演示应用。
